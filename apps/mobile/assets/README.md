# Assets

This directory contains app assets such as icons, splash screens, and images.

## Required Assets

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `splash.png` - Splash screen (1284x2778)
- `favicon.png` - Web favicon (48x48)

## Generating Assets

Use the Expo asset generation tools or provide your own custom assets.

```bash
# Generate icons and splash screens
npx expo-asset
```

For now, these are placeholders. Add your custom branding assets before building for production.
